# Sources

Use the Reaudit MCP tools to analyze citation sources and outreach:

1. Use `get_citation_sources` to see which pages AI models cite
2. Use `get_citation_analytics` for citation frequency and trends
3. Use `extract_author_info` to get contact info from a cited URL
4. Use `list_outreach_opportunities` to see potential link building contacts
5. Use `create_outreach_opportunity` to track a new outreach target

## Example prompts

- "Which of my pages do AI models cite most?"
- "Show me my top citation sources"
- "Extract the author info from this URL"
- "What are my outreach opportunities?"
- "Which pages should I optimize for more AI citations?"
