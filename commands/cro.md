# Conversion Rate Optimization

Use the Reaudit MCP tools to analyze and optimize conversion rates:

1. Use `get_cro_dashboard` for a unified view of conversion metrics
2. Use `list_conversion_goals` and `create_conversion_goal` to manage goals
3. Use `list_funnels` and `create_funnel` to build multi-step funnels
4. Use `get_funnel_analysis` to identify drop-off points
5. Use `list_ab_tests` and `get_ab_test_results` for experiment analysis
6. Use `list_landing_pages` and `analyze_landing_page` to score pages
7. Use `get_cro_recommendations` for AI-generated optimization suggestions
8. Use `list_cro_insights` and `update_cro_insight` to manage insights
9. Use `get_cro_strategy`, `create_cro_strategy`, or `update_cro_strategy` for strategy management
